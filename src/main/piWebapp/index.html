Welcome. It's here to serve the bsteele Music App used by CommunityJams.org in a live environment.

Notes:

  • It might be useful on a cell phone but you're much better off with a tablet or larger.
  • Be sure to set the screen timeout as long as possible to avoid re-starting the tablet often.
  • For best performance, close all other apps on your tablet/phone. This is particularly true on an ipad. Use Chrome (preferred) or Safari, and close the other (double-click home button, flick up on all apps but Chrome.
  • Bookmark this page or the bsteele Music App page itself.
  • Try the full screen button under the Options tab.
  • I would suggest you plug it into a power supply if you have one... you might be here a while!

Initial Setup:

Note that you won't find this page until you've done the below! Share this page with friends how haven't got the setup yet.

  1. On your tablet, go to Settings, Wifi
  2. Choose either "bob5" or "bob" Wifi. There will not be a password required. The bob5 wifi will be faster. You will likely be warned that there is no internet connection. Choose the options to allow no internet access for this wifi.
  3. Open your Chrome browser. Other browsers will likely work.
  4. Now the hard part: type: 192.168.0.205 into the URL location.
  5. You should now have this page. Click the bsteeleMusicApp link above.
